## 🚀 Lancer le projet

### Prérequis

- Avoir [Flutter installé](https://docs.flutter.dev/get-started/install)
- Un émulateur Android/iOS ou un appareil physique connecté

### Étapes

```bash
# 1. Cloner le dépôt
git clone https://github.com/TON_USERNAME/premier_projet.git

# 2. Entrer dans le dossier
cd premier_projet

# 3. Installer les dépendances
flutter pub get

# 4. Lancer l'application
flutter run
```
